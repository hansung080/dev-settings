# Assets
Various things and stuff.

## Vim
### Download .vimrc
To download `.vimrc` at your home directory, run:
```sh
curl -fsSL 'https://raw.githubusercontent.com/hansung080/assets/master/.vimrc' -o "$HOME/.vimrc"
```
